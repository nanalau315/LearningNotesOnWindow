####Rules for math order#####
- Parentesis
- Exponential
- Multiply
- Divide
- Addition
- Subtraction

- to comment, use control + /

####Special operator#####
- exponential power => **
- modulo => % 
    (10 % 3 = 1)
- integer division => // 
    (10//3 = 3) 
- it rounds down to the nearest decimal
- += plus equals
- -+ minus equal
